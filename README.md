# JLS Tecnologia - Landing Page

Landing page profissional para empresa de tecnologia especializada em suporte técnico, infraestrutura de TI e desenvolvimento de sistemas.

## 🚀 Tecnologias

- **Next.js 16** - Framework React com App Router
- **TypeScript** - Tipagem estática
- **Tailwind CSS 4** - Estilização moderna e responsiva
- **Lucide React** - Ícones modernos
- **Google Fonts** - Space Grotesk + DM Sans

## 📦 Instalação

```bash
npm install
```

## 🛠️ Desenvolvimento

```bash
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000) para ver o resultado.

## 🏗️ Build para Produção

```bash
npm run build
npm start
```

## ✨ Funcionalidades

### Seções da Landing Page

1. **Header** - Navegação fixa com logo JLS
2. **Hero Section** - Apresentação impactante com CTAs
3. **Client Logos** - Social proof com tipos de clientes
4. **Services** - 3 cards de serviços principais
5. **Infrastructure** - Destaque para infraestrutura com diagrama SVG
6. **Stats** - Contadores animados de estatísticas
7. **Process** - Timeline do processo de atendimento
8. **Development** - Seção de desenvolvimento de sites e sistemas
9. **Testimonials** - Depoimentos de clientes
10. **About** - Sobre a empresa e valores
11. **Contact** - Formulário integrado com WhatsApp
12. **Footer** - Links e informações de contato

### Recursos Técnicos

- ✅ Design responsivo (mobile-first)
- ✅ Animações suaves no scroll
- ✅ Contadores animados com IntersectionObserver
- ✅ Integração com WhatsApp no formulário
- ✅ SEO otimizado com metadata
- ✅ Performance otimizada (next/image, fonts)
- ✅ Acessibilidade WCAG AA
- ✅ Dark mode nativo

## 📝 Personalização

### Alterar informações de contato

Edite o arquivo `components/ContactSection.tsx` e `components/Footer.tsx`:

- WhatsApp: linha com `wa.me/5543999999999`
- E-mail: `contato@jlstecnologia.com.br`
- Telefone: `(43) 99999-9999`

### Alterar logo

Substitua o arquivo `public/images/logo.jpg` pela sua logo.

## 📱 Responsividade
